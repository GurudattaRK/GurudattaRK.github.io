PicoPerf

Zero-overhead PMU benchmarking library for x86-64 Linux. Captures hardware-native CPU metrics directly from x86-64 registers via perf_event_open — retired instructions, cycles, IPC, cache misses, TLB misses — with nanosecond resolution using RDTSCP.

TeraCrypt

Multi-core x86-64 assembly encryption engine compiled as a position-independent shared library. Python frontend distributes file chunks across physical cores with zero synchronization overhead, achieving 450-500% speedup on 6-core machines.
